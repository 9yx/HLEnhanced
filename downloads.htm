Downloads

Ready to empower your Half-Life 1 modding capabilities? Download Half-Life: Enhanced now!

Requirements

If you are planning on just using Half-Life: Enhanced, then you must have the Steam client installed and own a Steam account with a Half-Life 1 license. Steam must report that Half-Life 1 is "Ready to play". If you are planning to change/compile the source code, you might want to read the "Compiling" page on Half-Life: Enhanced's GitHub wiki.

Instructions

Select the version you want to use below, download the "Game" data by clicking on it's button. Extract the "hlenhanced" folder in your Half-Life installation. If you are a programmer, you can download the source code by clicking on it's button.

Note: all downloads are automatically generated. If you want to share any download, please post a link to this page instead of the direct downloads, thank you.

Stable

Stable versions have received a lot of testing, bug/crash fixes and features. They may get outdated quickly against the "Snapshot" and "Bleeding Edge" versions but if you are looking to create/upgrade your Half-Life 1 map/mod project, this is definetely the version you should use.

Snapshot

Snapshots versions contains more features but may not be stable as the "Stable" versions. If you are looking to test Half-Life: Enhanced to report bugs and/or you are not afraid of using a version that may contains bugs, then go ahead.

Bleeding Edge

If you want to get the very latest features and/or contribute to Half-Life: Enhanced's, this is the version you should use. Keep in mind that this version is likely to contain a lot of bugs and eventually crash. If you are looking to create/upgrade your Half-Life 1 map/mod project, we highly recommend to NOT use this version.
