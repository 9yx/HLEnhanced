About

Giving Half-Life 1 the love it deserves.

Half-Life: Enhanced is a project that give more power to Half-Life 1 modders for their creations.
The main features are cleaning Valve's code, provide better documentation (powered by Doxygen), bugs/crash fixes and a lot of new features.
Thanks to the open source model, anyone regardless of Half-Life 1 modding experience can contribute to the project.
Half-Life: Enhanced is a project created by Solokiller.
