Welcome to the GitHub pages of Half-Life: Enhanced!

Introduction

Half-Life: Enhanced (not to be confused with the mod on ModDB) is a project that aims to improve the lives of modders and players of Half-Life 1. Wanna learn more about it? Check out the "About" page.

Contributions

Want to contribute to the project itself or these pages? Feel free to do so by using the "GitHub Pages" branch or the "master" branch. Any contribution even if it's small is welcome.
