Detailed Description

This class is used to decide how to handle certain actions, and performs certain actions based on the active rules. An instance is created when a map starts. Gamerules exist for singleplayer, deathmatch, teamplay, and (in the future) co-op and CTF. TODO: co-op and CTF gamerules.
