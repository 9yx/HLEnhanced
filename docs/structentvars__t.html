Detailed Description

Contains entity variables that the engine needs to access for graphics, networking and physics.

Vector entvars_t::clbasevelocity

Base velocity that was passed in to server physics so client can predict conveyors correctly. Server zeroes it, so we need to store here, too. Zeroed out prior to DLL_FUNCTIONS::pfnPreThink being called, set to the player's basevelocity after the player's think has been called, before DLL_FUNCTIONS::pfnPostThink is called.
