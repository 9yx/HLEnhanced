29 //-----------------------------------------------------------------------------
30 // Purpose: Text-input handler
31 // Behaviour Specs:
32 // This class handles input from mouse and keyboard.
33 // TextEntry classes support several box styles, horizontal scrolling with no scrollbar
34 // vertical scrolling with or without a scrollbar, single line, multiline,
35 // editable and noneditable.
36 //
37 // Shared behaviour:
38 // URL's are a different text color and are clickable. Clicking them brings up a web browser.
39 // For vertical scroll bars, up and down arrows scroll one line at a time.
40 // Clicking and dragging the nob scrolls through text lines.
41 // Mouse wheel also moves the nob.
42 // User can select and highlight text in the window.
43 // Double clicking on a word selects it.
44 //
45 // Non editable:
46 // No blinking cursor in non editable windows.
47 // Right clicking mouse opens copy menu. Menu's top left corner is where mouse is.
48 // Ctrl-c will also copy the text.
49 // Editable:
50 // Blinking cursor is positioned where text will be inserted.
51 // Text keys type chars in the window.
52 // ctrl-c copy highlighted text
53 // ctrl-v paste highlighted text
54 // ctrl-x cut highlighted text
55 // ctrl-right arrow move cursor to the start of the next word
56 // ctrl-left arrow move cursor to the start of the prev word
57 // ctrl-enter delete the selected text (and inserts a newline if _catchEnterKey is true)
58 // insert delete selected text and pastes text from the clipboard
59 // delete delete the selected text
60 // ctrl-home move cursor to the start of the text
61 // ctrl-end move cursor to the end of the text.
62 // left arrow move cursor before prev char
63 // ctrl-shift left/right arrow selects text a word at a time
64 // right arrow move cursor before next char
65 // up arrow move cursor up one line.
66 // down arrow move cursor down one line.
67 // home move cursor to start of current line
68 // end move cursor to end of current line
69 // backspace delete prev char or selected text.
70 // Trying to move to the prev/next char/line/word when there is none moves the cursor to the
71 // start/end of the text.
72 // Horizontal scrolling:
73 // Trying to move to the prev/next char/line/word when there is none scrolls the text
74 // horizontally in the window so the new text displays at the correct side.
75 // When moving to prev chars scrolling is staggered. To next chars it is one char at a time.
76 // Cut/Copy/Paste Menu:
77 // Right clicking mouse brings up cut/copy/paste menu.
78 // If no text is highlighted the cut/copy options are dimmed. Cut is dimmed in non editable panels
79 // If there is no text in the clipboard or panel is not editable the paste option is dimmed.
80 // If the mouse is right clicked over selected text, the text stays selected.
81 // If the mouse is right clicked over unselected text, any selected text is deselected.
82 //
83 //
84 //-----------------------------------------------------------------------------
