Detailed Description

template<typename BASECLASS, typename DERIVEDCLASS>
class CASExtendAdapter< BASECLASS, DERIVEDCLASS >

Class used to adapt C++ classes for extension in Angelscript.

Template Parameters
BASECLASSC++ class to extend. Should be a class with at least one virtual function.
DERIVEDCLASSC++ class that will be instantiated. Used to call object methods correctly.
