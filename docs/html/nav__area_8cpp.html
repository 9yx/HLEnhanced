void GenerateNavigationAreaMesh ( void  )

This function uses the CNavNodes that have been sampled from the map to generate CNavAreas - rectangular areas of "walkable" space. These areas are connected to each other, allowing the AI to know how to move from area to area.

This is a "greedy" algorithm that attempts to cover the walkable area with the fewest, largest, rectangles.
