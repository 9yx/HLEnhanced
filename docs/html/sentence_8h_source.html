136 // A sentence can be closed captioned
137 // The default case is the entire sentence shown at start time
138 //
139 // "<persist:2.0><clr:255,0,0,0>The <I>default<I> case"
140 // "<sameline>is the <U>entire<U> sentence shown at <B>start time<B>"
141 
142 // Commands that aren't closed at end of phrase are automatically terminated
143 //
144 // Commands
145 // <linger:2.0> The line should persist for 2.0 seconds beyond m_flEndTime
146 // <sameline> Don't go to new line for next phrase on stack
147 // <clr:r,g,b,a> Push current color onto stack and start drawing with new
148 // color until we reach the next <clr> marker or a <clr> with no commands which
149 // means restore the previous color
150 // <U> Underline text (start/end)
151 // <I> Italics text (start/end)
152 // <B> Bold text (start/end)
153 // <position:where> Draw caption at special location ??? needed
154 // <cr> Go to new line
155
