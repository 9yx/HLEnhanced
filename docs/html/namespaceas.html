Creates a class declaration for an extension class that scripts can inherit from.

Parameters
pszClassNameThe class name that the class will have.
pszBaseClassNameClass or interface to inherit from. Can be a list of classes/interfaces, separated by commas.
pszCPPClassNameRegistered name of the class that represents the C++ version of the class. Will be represented by the self member.
pszCPPBaseClassNameRegistered name of the class that provides base class method calling features. Will be represented by the BaseClass member. Can be null, in which case no BaseClass member is defined.
pszClassContentsOptional. Contents to add to the class in addition to those provided by the framework.
Returns
String containing the class declaration.
